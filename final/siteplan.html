I Throw Paper - Site Plan

Site Name

I Throw Paper

The phrase, "I throw paper," while not typically announced in a casual game, draws on the instantly recognizable symbols of Rock, Paper, Scissors. I also feel like it is more approachable than using the acronym "RPS."

While not required, the domain ithrowpaper.com is available, as are .net, .xyz, .club, and all other TLDs I checked.

Site Purpose

This site is mostly intended to be used for entertainment. It will have some fun history and variations of Rock Paper Scissors, and the ability to play both a few different versions.

Scenarios

  • I'm bored and need to fill a couple minutes
  • Regular Rock, Paper, Scissors is too simple. How I can I change it up?
  • We all know Rock, Paper, Scissors, but where did it come from?

Color Schema

My primary color will be #BBD5ED. It will mostly be used as the background color for shared elements, like the site header and footer.

The first accent, or secondary color, is #720026. I plan on using this for page titles, section headers, and other textual navigation landmarks.

The second and third accent colors, still secondary colors, are #A288E3 and #FF9B54.

These will be used as alternating colors when there are multiple elements lined up, in conjunction with the primary color.

Typography

This is a more fun site, so there are two fonts being used. For headers, section titles, and other items that fit "textual landmarks" are in "". Everything else uses "".
